# ☕ Bespoke Cafe - Interactive Digital Menu

**Live Demo:** [https://shank078.github.io/bespoke-cafe-menu/](https://shank078.github.io/bespoke-cafe-menu/)

## 📖 Project Overview
This repository contains a localized, interactive digital menu built for a cafe/roastery. It is a reflectory demo of a freelance project originally delivered to a client in Toronto,
redesigned here to showcase functionality, UI/UX, and vanilla web development skills. 

The primary goal of this project was to solve a specific business problem: **Deploying a modern, interactive digital menu with absolutely zero ongoing hosting or subscription costs for the client.**

## 🛠️ Technical Stack
* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Modern styling utilizing CSS Variables (`:root`), Flexbox, CSS Grid, and custom keyframe animations.
* **Vanilla JavaScript:** DOM manipulation, array filtering, and dynamic modal generation without the overhead of heavy frameworks like React or jQuery.
* **Hosting:** Deployed as a static site via GitHub Pages for zero-cost, high-availability hosting.

## 🚀 Key Features
* **Zero Dependencies:** Built entirely with vanilla technologies to ensure fast load times and easy maintenance.
* **Interactive UI/UX:** Features a dynamic modal that visually breaks down the ingredients of each menu item using custom animations.
* **Mobile-First Design:** Fully responsive layout optimized for smartphone scanning via QR code.
* **Static Deployment:** Hosted on GitHub Pages, demonstrating an understanding of static site generation and serverless hosting models.
